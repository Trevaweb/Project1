• socket
	Creates a connection point for the client and server to connect to
• bind
	Bind the socket to an address on the server side
• listen
	Waits for connections to the socket on the server side
• accept
	accepts new sockets incoming from the client on ther server side
• connect
	uses the socket to connect to the server side
• send
	sends data from either the client to the server or the server to the client
• recv
	receives data from either the client or the server
